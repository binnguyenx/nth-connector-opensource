# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, email **jimmykhangnguyen@gmail.com** with:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

You can expect a response within 48 hours. Once the issue is confirmed and patched, you will be credited in the release notes (unless you prefer to remain anonymous).
